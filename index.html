Education

As my master thesis, I work on developing an attitude-position estimation algorithm for fusing data from an IMU, a GPS, and a Camera. To this end, I develop several fusing algorithms, including EKF and UKF. The designed navigation system will be used for control purposes in a quadrotor UAV.

B.Sc. in Mechanical Engineering

Isfahan University of Technology, Iran - 2014 - 2018

As my B.Sc. thesis, I had designed and implemented a navigation algorithm and a control algorithm for a quadrotor. The navigation algorithm was based on an adaptive complementary filter, and the control algorithm was a simple PID algorithm. I successfully managed to control an autonomous quadrotor using these algorithms.
